# 配置系统与生物交互

## 配置系统

### 架构

`ModEnchantmentConfig.java` — 所有附魔/能力参数集中管理。

```java
// 字段声明（按分组排列）
public static final DoubleValue DODGE_BASE_CHANCE;
public static final DoubleValue DODGE_CHANCE_PER_LEVEL;

// 静态块内 push/pop 初始化
static {
    BUILDER.push("闪避附魔配置");
    DODGE_BASE_CHANCE = BUILDER.comment("基础触发概率")
        .defineInRange("dodge_base_chance", 0.0, 0.01, 1.0);
    DODGE_CHANCE_PER_LEVEL = BUILDER.comment("从0级开始每1级增加的触发概率")
        .defineInRange("dodge_chance_per_level", 0.04, 0.01, 0.5);
    BUILDER.pop();
    // ...
}
```

### 关键约定

- **push/pop 顺序不可变更**：`BUILDER.push("分组名")` → 字段定义 → `BUILDER.pop()`，顺序错误会导致 TOML 子目录失效
- 所有倍率/概率/持续时间通过 `Config.SOME_KEY.get()` 获取，运行时自动生成 `mme_enchantments.toml`
- 配置注册：`ModLoadingContext.get().registerConfig(Type.COMMON, ModEnchantmentConfig.SPEC, "mme_enchantments.toml")`

## 生物交互机制

### 磁石传送 (`jizhi/TP/MagnetTeleportHandler.java`)

- 手持磁石右键 → 传送到绑定位置
- 冷却时间可配置（默认 150 秒）

### 铁傀儡目标 (`jizhi/Golem/IronGolemTargetHandler.java`)

- "呼唤来者"号角附魔 → 召唤铁傀儡攻击目标

### 雪傀儡优先目标 (`jizhi/Golem/SnowGolemPriorityTargetSystem.java`)

- 雪傀儡的雪球标记目标为优先攻击对象
- 臻冰效果：35% 概率触发额外伤害

### 唤魔者免疫 (`jizhi/Evoker/EvokerImmunityHandler.java`)

- 唤魔者 50% 概率免疫单次伤害

### 女巫药水 (`jizhi/Witch/WitchPotionHandler.java`)

- 女巫 50% 概率投掷大量药水

### 嗅探兽战利品 (`jizhi/Sniffer/LootTableEventHandler.java`)

- 嗅探兽挖掘时概率掉落特殊物品（经验瓶 50% / 光灵箭 40% / 附魔金苹果 10% / 嗅探兽蛋 15%）

### 骷髅射手游走 (`jizhi/cp/SkeletonSpeedHandler.java`)

- 骷髅 60% 概率在攻击时高速移动

### 末影人传送 (`jizhi/Enderman/EnderManTeleportHandler.java`)

- 末影人 30% 概率在被攻击时传送闪避

### 悦灵能力 (`jizhi/allay/MMEEntityHandler.java`)

- 周期触发特殊能力（间隔可配置，默认 12 秒）

### 发光效果伤害 (`jizhi/SA/GlowingEffectDamageHandler.java`)

- 发光状态下的亡灵生物每秒受到额外伤害
