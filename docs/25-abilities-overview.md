## 第 11 阶段：冒险者的觉醒

持有"冒险的终点"饰品时自动激活，无需开关。所有能力获得以下强化：

### 机制质变（10 个 🆕 标记能力）

| 能力 | 质变内容 |
|------|---------|
| 虚空踏步 | 二段跳 → 三段跳 |
| 灵魂绑定 | 死亡时经验值不掉落（经验球不生成） |
| 不朽装备 | 每件护甲 +1 护甲值，主手武器 +15% 伤害 |
| 翱翔 | 飞行速度 1.5x |
| 死亡抗拒 | 触发时自动释放一次免费旅者审判（无视冷却/GCD） |
| 影杀 | 斩杀时对 8 格内最多 16 只怪物削减 15% 影子血量 |
| 淬魂之力 | 目标 HP ≤ 20% 最大值时，真实伤害翻倍 |
| 嗜血 | 过量治疗转为吸收护盾（上限 20% 最大生命） |
| 净魂 | 周围 16 格内敌对生物自动获得虚弱 II（每 2 秒刷新） |
| 旅者之力 | 审判范围 +50%（9 格），庇护期间可 0.3x 速度移动 |

### 数值强化（15 个）

统一 ×1.3 倍率（可配置），特殊能力例外：

- 休养生息：额外 +2 amplifier（再生 V → VII）
- 鸿运当头：额外 +2（+3 → +5）
- 受击坚韧：额外 +6 层（12 → 18）
- 不动如山：×1.3 封顶 100%
- 灵巧：×1.3 封顶 95%
- 真实血量：修复容差翻倍
- 拒绝篡改：反弹 30% 被拦截伤害
- 环境免疫：标签匹配 → 全无源伤害免疫

## 实现方式速查

| 能力 | 实现机制 | 觉醒追加 |
|------|---------|---------|
| 灵巧 | `LivingAttackEvent` 概率 cancel | 倍率 ×1.3（封顶 95%） |
| 恩赐永驻 | PlayerTick 周期性延长 Buff | extendAmount ×1.3 |
| 大地之力 | `PlayerEvent.BreakSpeed` 乘算 | 倍率 ×1.3 |
| 虚空踏步 | 客户端跳跃输入 + 服务端多段跳逻辑 | 跳数追踪：2→3 段 |
| 休养生息 | PlayerTick 脱战检测 + `MobEffects.REGENERATION` | amplifier +2 |
| 灵魂绑定 | `LivingDeathEvent` 保存 / `PlayerEvent.Clone` 恢复 | 死亡前清零经验防掉落 |
| 不动如山 | `Attributes.KNOCKBACK_RESISTANCE` setBaseValue | 倍率 ×1.3（封顶 100%） |
| 伤害抗性 | `LivingHurtEvent` 乘算减伤 | 倍率 ×1.3（封顶 95%） |
| 无形之手 | `ForgeMod.BLOCK_REACH` setBaseValue | 倍率 ×1.3 |
| 不朽装备 | Mixin 拦截耐久消耗 | 护甲/武器 AttributeModifier |
| 鸿运当头 | `LootingLevelEvent` + `FortuneFavorMixin`（EnchantmentHelper） | 固定 +2 级（含 FortuneContext 觉醒标记） |
| 环境免疫 | `LivingAttackEvent` 按 DamageTypeTags 拦截 | 觉醒：全无源伤害免疫 |
| 嗜血 | `LivingHurtEvent` 攻击方回血 | ×1.3 + 过量 → 吸收护盾 |
| 禁疗之触 | `LivingHurtEvent` 施加 UndyingSlashEffect | 时长 ×1.3 |
| 坚韧之躯 | `Attributes.MAX_HEALTH` setBaseValue | ceil(×1.3) |
| 受击坚韧 | `LivingHurtEvent` 叠层 + 减伤，PlayerTick 超时归零 | +6 层（ResilienceAbility 重载） |
| 净魂 | PlayerTick 清除 HARMFUL 效果 | 每 2 秒周围怪物虚弱 II |
| 翱翔 | PlayerTick 维持 `mayfly = true` | flyingSpeed 1.5x（一次性设，不乘爆） |
| 淬魂之力 | `LivingHurtEvent` + 自定义 DamageSource 穿透 | HP≤20% 时伤害 ×2 |
| 破敌之眼 | Mixin 拦截 `isInvulnerableTo` + `actuallyHurt` | 无敌帧目标 +30% 伤害 |
| 死亡抗拒 | `LivingDeathEvent` 取消死亡 + Mixin 无敌期双层拦截 | 触发时调用 executeJudgment() |
| 影杀 | `LivingHurtEvent` 影子血量 NBT + 饱和式秒杀 | 斩杀时 8 格 AOE |
| 真实血量 | Mixin 备份/修复血量 | 容差 epsilon ×2 |
| 拒绝篡改 | Mixin 拦截 `setHealth` + maxHealth 底线 | 反弹 30% 给 lastHurtByMob |
| 旅者之力 | 客户端按键 + 网络包 + AOE 伤害 / 无敌效果 | 审判 ×1.5 范围 + 庇护可移动 |

## 数值配置

所有倍率、概率、持续时间均可通过 `adventure_power.toml` 调整。

配置路径：
- 能力数值：`冒险能力配置 > 能力数值 > [能力名称]`
- 觉醒强化：`冒险能力配置 > 能力数值 > 觉醒强化 > [觉醒项]`

觉醒全局倍率：`awaken_multiplier`（默认 1.3）
