# 物品与装备

## 冒险饰品（Curios body slot）

两个饰品是新模组仅有的物品。原作者的物品（超越晶剑、下界绝魔刀、地狱图腾等）全部不留，需要时再重新设计。

| 物品 | 配方 | 效果 |
|------|------|------|
| **冒险的开始** | 自定义合成（待设计） | 激活 adventurer，开始追踪里程碑 |
| **冒险的终点** | 龙蛋 + 鞘翅（AdventureEndRecipe） | 全解锁，开启 7 种高级能力 |

### 设计约束

- 两个饰品均仅创造模式可卸（`canUnequip()` 硬编码）
- "不离不弃"附魔对此不生效（饰品自身的卸除保护独立于附魔）
- 当前渲染模型借用了第三方 3D 模型，需要替换为自制纹理
- 临时方案：先制作 2D 平面纹理（16×16），后续可升级为 3D 模型

### 纹理设计（暂定）

| 物品 | 配色 | 设计思路 |
|------|------|---------|
| 冒险的开始 | 蓝色/青色 | 象征起点的简洁符号（星形 / 罗盘 / 传送门碎片） |
| 冒险的终点 | 金色/紫色 | 象征完成的符号（完整星形 / 龙蛋核心 / 终焉之眼） |
