# 冒险的力量 — 25 种能力总览

> 力量属于冒险者自身。以下按里程碑解锁顺序列出全部 25 种能力。

## 能力分类说明

| 分类 | 说明 |
|------|------|
| ⚔️ 战斗 | 直接提升作战能力 |
| 🛡️ 防御 | 减伤、抗性、免死 |
| 🔧 采集 | 挖掘、时运、触及 |
| ✨ 辅助 | Buff 管理、飞行、经验保留 |
| 🩸 机制 | 血量保护、禁疗、真实伤害 |

## 觉醒机制

- **10 个能力**获得**机制质变** — 新增特效或改变运作方式（标注 ⭐）
- **15 个能力**获得**数值强化** — 基础值 × 觉醒倍率（默认 1.3x，标注 📈）
