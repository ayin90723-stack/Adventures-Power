# 主动技能与 UI 系统

## 主动技能系统

### 概述

FullyUnlocked 级能力，两个主动技能共享 GCD（全局冷却，默认 0.5 秒）。技能通过按键触发：
- **R 键**：释放当前选中技能
- **T 键**：切换技能（审判 ↔ 庇护）

### 旅者审判 (Judgment)

**AOE 伤害技能**，以玩家为中心范围伤害。

#### 伤害公式

```
总伤害 = 基础固伤(10)
       + 目标 maxHealth × (3% × 里程碑数)  // 满里程碑 = 15%
       + 目标 currentHealth × (3% × 里程碑数)
```

- 范围：默认 6 格
- 冷却：默认 30 秒（600 tick）
- 只对敌对生物生效（Monster + 非驯服 + 排除友好火力保护目标）

#### 兜底机制

`hurt()` 被外部 mod 拦截时 → 用 `HealthUtil.setAllHealthLikeRaw()` 补扣 → 仍失败 → 直接调用 `die()`。

### 旅者庇护 (Sanctuary)

**无敌技能**，类似图腾但更强大：

- 无敌持续时间：默认 5 秒（100 tick）
- 冷却：默认 90 秒（1800 tick）
- 触发时：清除所有负面效果 + 灭火 + 图腾粒子特效
- 无敌期间：`PlayerStateHandler.onSanctuaryDamage` 以 HIGHEST 优先级取消 `LivingAttackEvent`（优先于其他防御；`BYPASSES_INVULNERABILITY` 如 /kill、虚空不拦截）
- 移动限制：非觉醒期间移动速度锁定为 0；觉醒后可以 30% 速度移动（`AWAKEN_SANCTUARY_SPEED`）

### 数据存储

所有技能状态存储在 `IAdventureProgress` Capability 上：

| 字段 | 类型 | 说明 |
|------|------|------|
| `activeSkillIndex` | int | 0 = 审判, 1 = 庇护 |
| `judgmentCooldownEnd` | long | 审判冷却结束 tick |
| `sanctuaryCooldownEnd` | long | 庇护冷却结束 tick |
| `sanctuaryInvulEnd` | long | 庇护无敌结束 tick |
| `activeSkillGcdEnd` | long | GCD 结束 tick |

### 网络通信

- 客户端按键 → `NetworkHandler.SkillReleasePacket` → 服务端 `ActiveSkillHandler.handleSkillRelease()`
- 冷却状态通过 Capability 同步到客户端，HUD 显示就绪/冷却状态

## HUD 叠加层

### 主动技能 HUD (`ActiveSkillHudOverlay`)

- 位置：屏幕右上角
- 显示：当前选中技能名称 + 是否就绪
- 冷却中显示剩余秒数

### 死亡抗拒 HUD (`DeathDefyHudOverlay`)

- 无敌期间：全屏金色边框（ARGB: 0x50FFD700，2px 宽）
- 中上方：金色倒计时

## 面板系统

### Buff 管理面板 (`BuffManagementScreen`)

**O 键打开。**

- 列出玩家当前所有正面效果
- 每个效果显示名称 + 剩余时间 + 状态（永续 / 正常到期）
- 单击切换：永续 ↔ 正常到期
- 永续效果由"恩赐永驻"能力维持（每 3 秒检查，低于保底值自动续期）
- 排除列表存储在 `player.persistentData.MME_BuffBlacklist` 中

### 冒险之力面板 (`AdventureEndAbilityScreen`)

**P 键打开。**

- 显示 9 种能力及其启用/禁用状态
- 单击切换开关
- 能力按 `KNOWN_ABILITIES` 注册表顺序显示
- 未达到 `fullyUnlocked` 的能力显示为灰色不可操作
- 数据来源：客户端 Capability 缓存（通过 `AdventureSyncPacket` 同步）
- 打开前自动请求同步（`requestSyncAndOpenScreen(PENDING_ABILITY)`）

## 按键绑定

| 按键 | 功能 | 分类 |
|------|------|------|
| O | 打开 Buff 管理 | MME |
| P | 打开冒险之力管理 | MME |
| R | 释放主动技能 | 游戏 |
| T | 切换主动技能 | 游戏 |
