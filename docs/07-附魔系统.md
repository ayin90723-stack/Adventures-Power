# 附魔系统

## 核心附魔（8 个，全部自研）

新模组只保留以下 8 个我们原创或深度重写的附魔。原作者的附魔全部不留。

---

## 战斗附魔

### 见既斩 — 穿透一切无敌

| 属性 | 值 |
|------|-----|
| 适用 | 武器 |
| 门禁 | FullyUnlocked（佩戴冒险的终点） |
| 归属 | **独创** |

4 层 Mixin 穿透架构：`Player.attack()` → `isInvulnerableTo` → `hurt()` 拦截 → 兜底 `actuallyHurt` + `die()`。配合 `InvulClearUtil` 反射清除其他 mod 的自定义无敌字段。

详见 [03-战斗机制.md](03-战斗机制.md#见既斩--穿透一切无敌)

---

### 不死斩 — 禁疗与反复活

| 属性 | 值 |
|------|-----|
| 适用 | 武器 |
| 门禁 | 无 |
| 归属 | **独创** |

攻击施加"不死斩印记"Buff：禁止目标回血（tick 末钳制血量到追踪的低点）、禁止目标复活。Mixin 双层拦截（setHealth + tick TAIL 字段直写反制）。

与淬魂联动：有不死斩标记的目标淬魂伤害 ×1.5。

详见 [03-战斗机制.md](03-战斗机制.md#不死斩--禁疗与反复活)

---

### 淬魂 — 多维伤害加成

| 属性 | 值 |
|------|-----|
| 适用 | 武器 |
| 门禁 | 末影龙里程碑 |
| 归属 | **完全重写** |

伤害公式：
```
总伤害 = 基础固伤 × 等级
       + 目标最大生命值 × 0.5% × 等级
       + 目标当前生命值 × 0.5% × 等级
       × (有不死斩标记 ? 1.5 : 1.0)
```

详见 [03-战斗机制.md](03-战斗机制.md#淬魂--多维伤害加成)

---

### 淬魂Plus — 影子血量独立斩杀

| 属性 | 值 |
|------|-----|
| 适用 | 武器 |
| 门禁 | FullyUnlocked（佩戴冒险的终点） |
| 归属 | **独创** |
| 对玩家 | 无效 |

在目标 NBT 中维护独立的"影子血量"，每次攻击按百分比 + 固伤削减。归零时触发三层斩杀路径（`hurt()` → `die()` → `saturationKill()`），无视一切防御。

详见 [03-战斗机制.md](03-战斗机制.md#淬魂plus--影子血量独立斩杀)

---

### 闪避 — 概率免疫伤害

| 属性 | 值 |
|------|-----|
| 适用 | 通用（护甲） |
| 门禁 | 下界里程碑 |
| 归属 | **深度修改** |

从原版的"拦截伤害"改为**拦截整个 `hurt()`** 调用。基础概率 0%（1 级 4%），每级 +4%，触发时完全免疫该次伤害。

## 特殊附魔

### 永恒 — 物品永不损坏

| 属性 | 值 |
|------|-----|
| 适用 | 任意物品 |
| 门禁 | 无 |
| 归属 | **独创** |

通过 `LegacyHurtAndBreakMixin` 拦截 `ItemStack.hurtAndBreak()`，禁止装备耐久消耗。

---

### 传承 — 受击累积增强

| 属性 | 值 |
|------|-----|
| 适用 | 任意物品 |
| 门禁 | 无 |
| 归属 | **重写** |

与原版差距很大。每次受击积攒进度，进度满后获得传承层数增加生命值。也受 `LegacyHurtAndBreakMixin` 保护（禁止耐久消耗）。

---

### 不离不弃 — Curios 反扒

| 属性 | 值 |
|------|-----|
| 适用 | 装备 |
| 门禁 | 无 |
| 归属 | **独创** |

两条 Mixin 拦截路径覆盖 Curios 物品的所有卸除方式：
- `CuriosExtractMixin`：拦截 `DynamicStackHandler.extractItem()`
- `ItemStackHandlerMixin`：拦截 `setStackInSlot()` 覆写路径

附加 `onCurioUnequip()` 事件层补充防御。冒险饰品自身 `canUnequip()` 硬编码仅创造可卸，独立于此附魔。

---

## 附魔与门禁关系

```
无门禁（开局可用）
├── 不死斩
├── 永恒
├── 传承
└── 不离不弃

里程碑解锁（佩戴冒险的开始后）
└── 下界 → 闪避
└── 末影龙 → 淬魂

FullyUnlocked（佩戴冒险的终点后）
├── 见既斩
└── 淬魂Plus
```
