# 项目总览

## 定位

**暂定名：冒险的力量（Adventure's Power）**

一个 Minecraft Forge 1.20.1 模组，以"冒险饰品"为入口，在玩家达成原版进度里程碑的过程中逐步解锁超模附魔与被动能力。

## 起源

本项目起源于 [我的更多附魔 (MyMoreEnchantments)](https://github.com/MC-XingKong/MyMoreEnchantments) v1.2.3 的自用修改版（原作者 MIT 协议）。经过大量改造后，**现已与原版有本质区别** — 不再是"魔改版"，而是一个独立的模组。

### 与原版的核心区别

| 维度 | 原版 mme 1.2.3 | 本项目 |
|------|---------------|--------|
| 附魔数量 | 30 | 8（全自研/深度重写） |
| 超模附魔 | 无限制可用 | 冒险进度门禁解锁 |
| 进度系统 | 无 | 5 里程碑 → 3 层 Capability 状态模型 |
| 主动技能 | 无 | 旅者审判 / 旅者庇护 |
| Mixin 注入 | 0 | 11 个 |
| 血量安全 | 无 | 真实血量 + 拒绝篡改 + 死亡抗拒 |
| Curios 集成 | 无 | body slot 饰品 + 不离不弃反扒 |
| 配置系统 | 硬编码 | 50+ 配置项自动生成 TOML |
| UI | 无 | Buff 管理 / 冒险之力 / HUD 叠加层 |
| 武器进化 | 无 | 超越晶剑进化 + 下界绝魔刀 |
| 淬魂 | 原版简单伤害加成 | 完全重写，与原版无关系 |
| 淬魂Plus | 不存在 | 独立影子血量斩杀系统 |
| 见既斩 | 不存在 | 多层 Mixin 穿透一切无敌 |
| 不死斩 | 不存在 | 禁疗 + 反复活 |

## 设计理念

### 门禁驱动的解锁体系

超模附魔不是开局就能用的。玩家需要：

1. **佩戴"冒险的开始"**（Curios body slot）→ 成为"冒险者"，解锁基础超模附魔
2. **达成 5 个原版进度里程碑**（下界/凋零/监守者/末影龙/鞘翅）→ 逐步解锁更多附魔
3. **佩戴"冒险的终点"** → 永久完全解锁，包括 7 种高级被动能力

### "饰品是钥匙，力量属于玩家"

冒险饰品不是装备 — 佩戴一次即永久激活对应状态（不可逆）。数据存储在玩家 Forge Capability 上，饰品只是触发媒介。

### 防御性设计

在高度模组化的环境中，其他模组可能通过多种方式干扰玩家血量、装备、无敌状态。本项目通过 Mixin 注入构建了多层防御体系，确保核心机制不受外部干扰。

## 技术栈

- **Minecraft**: 1.20.1
- **Forge**: 47.4.10
- **Java**: JDK 17
- **Mixin**: SpongePowered Mixin 0.8.5
- **依赖**: Curios API 5.14.1+

## 模块总览

```
冒险的力量/
├── 冒险饰品系统/          # Capability 驱动的进度与能力
│   ├── 3 层状态模型        # 激活层 → 里程碑层 → 查询层
│   ├── 9 种可开关能力      # 恩赐永驻/虚空踏步/死亡抗拒/伤害抗性/环境免疫/受击坚韧/拒绝篡改/真实血量/旅者之力
│   └── 2 个 Curios 饰品    # 冒险的开始 / 冒险的终点
│
├── Mixin 注入体系/         # 11 个 Mixin（详见 05-Mixin注入体系.md）
│   ├── 见既斩 x3           # 穿透一切无敌
│   ├── 不死斩 x1           # 禁疗 + 血量低点追踪
│   ├── 真实血量 x1         # Capability 备份 → 自动修复
│   ├── 拒绝篡改 x2         # 拦截非法血量修改
│   ├── 死亡抗拒 x1         # 双层拦截取消死亡
│   ├── 不离不弃 x2         # Curios 反扒
│   └── 永恒装备 x1         # 禁止耐久消耗
│
├── 战斗机制/
│   ├── 见既斩               # 4 层穿透：Player.attack → isInvulnerableTo → hurt 拦截 → 兜底穿透
│   ├── 不死斩               # 禁疗 Buff + tick 末血量钳制
│   ├── 淬魂                 # 完全重写：固定 + maxHP% + currentHP% + 不死斩标记增伤
│   ├── 淬魂Plus             # 影子血量独立追踪 → 三层斩杀路径（hurt/die/saturationKill）
│   └── 闪避                 # 拦截整个 hurt() 调用（门禁：下界里程碑）
│
├── 血量安全系统/
│   ├── 真实血量             # Capability 独立备份，不受 SynchedEntityData 污染
│   ├── 拒绝篡改             # HURT_DEPTH 标记 + AttributeInstance 所有者追踪
│   └── 死亡抗拒             # 冷却制免死 + 无敌期 + 自动清负面效果
│
├── 主动技能系统/
│   ├── 旅者审判             # AOE 伤害（基础固伤 + 目标 maxHP% + currentHP%）
│   ├── 旅者庇护             # 无敌 + 清负面 + GCD
│   └── HUD 叠加层           # 技能状态显示
│
├── UI 系统/
│   ├── Buff 管理面板        # O 键打开，切换每个正面效果的永续/正常状态
│   ├── 冒险之力面板         # P 键打开，9 种能力独立开关
│   └── 死亡抗拒 HUD         # 无敌期金色边框 + 倒计时
│
├── 物品与装备/
│   ├── 冒险的开始             # Curios body slot 饰品，激活冒险者身份
│   └── 冒险的终点             # 龙蛋 + 鞘翅合成，完全解锁所有能力
│
└── 配置系统/
    └── 精简配置参数            # 仅保留 8 附魔 + 冒险能力相关配置
```
