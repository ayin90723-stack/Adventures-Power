## 概述

**冒险的力量** 是一个 Minecraft Forge 1.20.1 模组，以 **冒险旅程驱动能力成长**。力量不来自装备附魔，不来自等级数值——力量源于你走过的路、击败的敌人、征服的维度。

- 🎯 **10 个里程碑** — 从度过第一夜到获得鞘翅
- ⚡ **26 种内在能力** — 闪避、飞行、吸血、免死、真实伤害……全部可自由开关
- 🌟 **第 11 阶段「冒险者的觉醒」** — 14 个机制质变 + 11 个数值强化
- 🔗 **与原版成就双向联动** — 完成原版成就自动完成里程碑，反之亦然
- 🎛️ **全能力自由开关** — 按 K 键打开面板，想用什么自己决定
- ⚙️ **高度可配置** — 所有倍率、概率、持续时间均可在 TOML 配置文件中调整

## 冒险旅程

### 冒险饰品：旅程的见证者

首次登录时自动获得 **「冒险的开始」**，佩戴在 Curios 身体栏位。饰品与灵魂绑定，不可卸除（创造模式除外）。右键饰品可随时查看里程碑进度和已解锁能力。

```
玩家首次登录
  ↓
自动获得「冒险的开始」→ 激活冒险者身份
  ↓
逐步完成 10 个里程碑（在成就界面可查看路线）
  ↓
全部里程碑达成 →「冒险的开始」进化为「冒险的终点」
  ↓
激活「冒险者的觉醒」—— 全部 26 种能力 + 觉醒强化
```

## 操作指南

| 按键 | 功能 |
|------|------|
| **K** | 打开「冒险之力」面板 — 查看和开关所有已解锁能力 |
| **B** | 打开「Buff 管理」面板 |
| **V** | 切换主动技能（旅者审判 ⇄ 旅者庇护） |
| **G** | 释放当前选中的主动技能 |
| **空格（空中）** | 虚空踏步 — 二段跳（觉醒后三段跳） |

所有按键可在原版「控制」设置中自定义。

### 主动技能

- **旅者审判**：对周围 6 格（觉醒 9 格）所有生物造成范围伤害（固伤 + 目标生命百分比），冷却 30 秒
- **旅者庇护**：短暂无敌 5 秒 + 清除所有负面效果，冷却 90 秒。觉醒后可缓慢移动（0.3x 速度）

## 安装

1. 安装 Minecraft Forge 1.20.1
2. 下载并安装 [Curios API](https://www.curseforge.com/minecraft/mc-mods/curios)
3. 将 `adventure_power-1.1.1.jar` 放入 `mods` 文件夹
4. 启动游戏，进入世界后自动获得「冒险的开始」

## 从源码构建

```bash
git clone https://github.com/ayin90723-stack/Adventures-Power.git
cd Adventures-Power
./gradlew build -x test
# 构件：build/libs/adventure_power-1.1.1.jar
```

**要求**：JDK 17，Curios API（放入 `libs/` 目录或通过 CurseMaven 配置）。

## 配置文件

所有数值均可通过 `config/adventure_power.toml` 调整：

- 各能力的倍率、概率、持续时间
- 觉醒全局倍率（`awaken_multiplier`，默认 1.3）
- 各能力的觉醒独立配置项

使用 `/reload` 或重新进入世界即可热更新配置。

## 常见问题

**Q: 能力太多了，可以关掉一部分吗？**
A: 可以。按 K 打开面板，每个能力旁边都有开关。

**Q: 死了 Buff 和经验会丢吗？**
A: 开启「灵魂绑定」后，死亡时保留全部正面 Buff 和经验值。觉醒后经验也完全不掉落。

**Q: 和其他模组兼容吗？**
A: 核心机制通过 Mixin 在底层拦截，设计上考虑了模组环境兼容性。遇到冲突欢迎反馈。

**Q: 可以在服务器使用吗？**
A: 可以。能力逻辑全部在服务端执行，客户端仅用于 UI 显示和按键输入。
